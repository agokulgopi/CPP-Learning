# C++ Practice Programs

Collection of basic C++ programs for learning.

## How to Compile and Run

### On Windows:
```bash
g++ simple_interest.cpp -o simple_interest
simple_interest.exe
```

### On Linux/Mac:
```bash
g++ simple_interest.cpp -o simple_interest
./simple_interest
```

## Requirements
- C++ compiler (GCC, MinGW, or MSVC)
